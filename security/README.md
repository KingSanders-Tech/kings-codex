# Security

Manual audits, hardening procedures, and triage commands for the systems I work on. Mostly Mac focused for now since that's my daily driver, but Linux notes will land here too as I build them out.

## Index

- [Mac Audit](./mac-audit.md) - Manual security audit for macOS. Catches ghost users, suspicious launch agents, unexpected network connections, and scheduled tasks. Run monthly, diff against baseline.
- [Authorization and Scope](./authorization-and-scope.md) - Notes on authorization, scope, and the difference between permission to scan and permission to access. Built from a real session on a friend's network.
- [Network Basics](./network-basics.md) - Foundational networking concepts every recon tutorial assumes you know. Private vs public IPs, CIDR notation, NAT, and how to find your network range before scanning.
